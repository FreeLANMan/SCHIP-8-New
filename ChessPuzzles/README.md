Jogo de quebra cabeças de xadrez com criador de novos desafios em HTML5 para SuperChip.

- O game contém 10 desafios.

- Use "E" para selecionar a peça e "E" para definir a casa para onde ela deve ser movida.

- O game está disponível aqui nos formatos .8o (Octo), ch8 (emuladores SCHIP) e HTML (para navegador).

- Criado usando Gemini 3.1 Pro grátis.

- Criado usando como base o game Slippery Slope de John Earnest.

- Puzzles e revisão dos puzzles pelo site https://lichess.org/
