# SCHIP-8-New
New stuff for CHIP-8 and SCHIP.

Novos conteúdos para CHIP-8 e Super CHip (os emuladores mais simples que existem!)

## Mas por quê?
Eu apoio uma informática de baixo custo e baixo impacto ambiental. CHIP-8 roda em qualquer máquina desde os anos 1970, incluindo máquinas alternativas de baixo custo baseadas em microcontroladores.

## O que você vai encontrar aqui?
App e games para CHIP-8 e SCHIP, ferramentas em HTML5, etc. Coisas relacionadas à esses sistemas.
